Inpainting-Based Hairstyle Transfer via Region-Specific Diffusion
SNOW Corp. 연구 인턴  ·  2025.10 — 2026.04 (6개월)
문제 정의

Hairstyle Transfer는 source와 reference 이미지 간 두부 자세(head pose) 차이가 클 때 특히 어렵습니다. 헤어는 비강체(non-rigid)이며 자세에 따라 실루엣이 크게 달라지기 때문에, 기존 방법들은 (1) 부정확한 편집 영역 설정으로 인한 오류, (2) reference 헤어의 세밀한 질감과 구조 재현 실패라는 두 가지 한계를 보였습니다. 이를 해결하기 위해 source pose에 정렬된 공간적 가이던스가 필요합니다.

핵심 아이디어
Region-Specific Loss

헤어 영역과 비헤어 영역에 서로 다른 학습 목표를 적용하여, 레퍼런스 조건이 헤어 영역에만 반영되도록 제어했습니다. 이를 통해 크로스 어텐션이 공간적으로 분리되며, 특정 영역만 변경되는 편집을 구현했습니다.

Source-Aligned Attention Mask

모델의 어텐션을 활용해, 소스 이미지의 자세에 정렬된 헤어 마스크를 자동으로 추출합니다. 별도의 마스크 입력 없이도 헤어 영역을 구분할 수 있습니다.

Coarse-to-Fine Inpainting

Attention 기반의 coarse 마스크로 중간 이미지를 생성한 후, 픽셀 수준의 마스크를 다시 추출하여 최종 인페인팅에 활용합니다.

결과
Qualitative Results. 소스 인물의 정체성과 기타 시각적 요소를 유지한 채, 레퍼런스 헤어스타일을 자연스럽게 반영한 결과입니다.
확장성 및 활용

텍스트 기반 헤어 속성 제어를 지원하며, IP-Adapter 기반 구조를 활용해 FaceID Plus와 결합한 설정도 함께 사용할 수 있습니다. 또한 Gradio 인터랙티브 데모와 ComfyUI 커스텀 노드로 추론 파이프라인을 배포했습니다.

MINDiff: Mask-Integrated Negative Attention for Controlling Overfitting in Text-to-Image Personalization ▼
문제 정의

대표적인 personalization 모델인 DreamBooth는 소수의 레퍼런스 이미지로 학습할 때 과적합 문제가 발생합니다. 기존의 Prior-Preservation Loss는 추가 데이터와 학습 비용이 필요하며, 추론 단계에서 유연한 제어가 어렵습니다. 이러한 문제를 identifier token의 영향으로 텍스트 프롬프트가 약화되는 prompt dilution 현상으로 해석했습니다.

핵심 아이디어
Negative Attention

Cross-Attention에서 학습된 reference 이미지의 영향을 선택적으로 감소시키는 메커니즘을 도입하여, identifier token에 의해 강화된 객체 정보를 억제하고 텍스트 프롬프트의 영향이 반영되도록 했습니다.

Mask-Integrated 공간 제어

Identifier token의 attention map을 활용해 subject 영역을 자동으로 추출하고, 이를 기반으로 subject와 그 외 영역을 구분했습니다. 억제 메커니즘은 subject 이외의 영역에만 적용되도록 설계하여, subject는 유지하면서 텍스트 조건이 반영되도록 했습니다.

λ 파라미터 제어

추론 시 λ를 조절하여 subject 충실도와 텍스트 정합성 간 균형을 유연하게 제어합니다. 기존 방식이 학습 단계에서 고정되는 것과 달리, 재학습 없이 즉시 조정 가능합니다.

Training-free

모델 구조나 가중치를 변경하지 않고 추론 단계에서 동작합니다. 기존에 학습된 DreamBooth 모델에 즉시 적용 가능하며 SD 1.4, SD 2.1, SDXL 등 다양한 백본에서 검증되었습니다.

결과
Qualitative Results. subject는 유지하면서, 텍스트 조건이 더 잘 반영되도록 한 결과입니다.

Controlling Overfitting in Text-to-Image Personalization for Diffusion Models ▼
경북대학교 컴퓨터학부 석사학위논문

Text-to-Image Personalization 과정에서 소수의 레퍼런스 이미지로 인해 발생하는 과적합 문제를 연구했습니다. 크로스 어텐션에 마스크 기반 Negative Attention을 도입한 MINDiff를 제안하여, 별도 재학습 없이 추론 단계에서 과적합을 완화하는 방법을 제안했습니다. 또한 IP-Adapter에서 나타나는 생성 다양성 감소 현상을 분석하고, 생성 단계(timestep)에 따라 어텐션 가중치를 동적으로 조절하는 기법(Dynamic Adjustment)을 제안해 텍스트 기반 창의적 이미지 생성 능력을 회복했습니다.

문제 정의

Text-to-Image 모델은 다양한 결과를 생성할 수 있음에도 불구하고, IP-Adapter를 사용할 경우 random seed를 변경해도 유사한 결과가 반복되는 문제가 발생합니다. 이를 텍스트와 이미지 조건 간 간섭(entanglement)으로 인해 생성 다양성이 제한되는 현상으로 해석했습니다.

핵심 아이디어
Time-Step별 조건 제어

diffusion의 생성 과정이 coarse-to-fine 단계로 진행된다는 점에 착안하여, 추론 time step에 따라 텍스트와 이미지 조건의 가중치를 다르게 조절했습니다.

Diversity 향상

초기 단계에서는 텍스트 조건의 영향을 강화하고, 이후 단계에서는 이미지 조건을 반영함으로써, 조건 간 균형을 유지하면서 생성 다양성을 확보했습니다.

Training-free

모델 구조를 수정하지 않고 추론 단계에서만 적용되며, 기존 IP-Adapter 기반 모델에 바로 사용할 수 있습니다.

결과
Qualitative Results. λ 값에 따라 텍스트 프롬프트와 레퍼런스 이미지 조건의 반영 정도를 조절한 결과입니다.
